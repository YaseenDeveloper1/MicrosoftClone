# Microsoft Clone

Built with **HTML, CSS, and JavaScript**, this project is a **fully responsive clone of Microsoft’s official website**. It features a **mobile-first design**, interactive navigation, smooth hover effects, and a modern UI, optimized for **all screen sizes**.

## Key Features
- Fully responsive layout using CSS Flexbox and Grid
- Interactive navigation bar and dropdown menus
- Smooth hover effects and transitions
- Modern UI inspired by Microsoft’s official website
